    Layouts.FlexLayout Class

    Extends Core.View
    Defined in: samsara/layouts/FlexLayout.js:25

    A layout which arranges items vertically or horizontally within a containing size. Items with a definite size in the specified direction keep their size, where items with an undefined size in the specified direction have a flexible size. Flexible sized items split up the left over size relative to their flex value.

    Defines the rules for composing layout specs: transform, align, origin and opacity. Transform is multiplied by the parent's transform (matrix multiplication). Align is a proportional offset relative to the parent size. Origin is a proportional offset relative to the current size. Opacity is multiplied by the parent's opacity.
